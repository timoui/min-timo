# wxc-panel

> 面板 - 小程序组件

## Install

``` bash
$ min install @timoui/wxc-panel
```


## API

### Panel

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`title`           | 面板标题 默认 无        |
|`border`         | 面板边框, 默认 true |
|`style`         | 面板额外样式, 默认 无 |
