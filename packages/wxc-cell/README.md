# wxc-cell

> 单元格 - 小程序组件

## Install

``` bash
$ min install @timoui/wxc-cell
```


## API

### Cell

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`slot:bd`           | 左侧信息         |
|`slot:ft`         |  右侧信息       |
|`access`         |  右侧箭头, 默认 `false` |
|`last`         |  最后一个列表, 默认 `false` |
